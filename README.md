# UG吃啥 - UniApp微信小程序

## 项目简介

"UG吃啥"是一个帮助用户决定吃什么的微信小程序，通过随机抽取的方式来解决选择困难症。

## 功能特性

### 1. 智能抽签
- 🎲 随机抽取今天吃什么
- 🎯 支持快速筛选（早餐、午餐、晚餐等）
- ⚙️ 高级筛选设置
- 🎨 精美的抽签动画效果

### 2. 粮库管理
- 📝 自定义菜品菜单
- 🏷️ 标签分类（早餐、午餐、晚餐、夜宵、零食）
- 💰 价格设置
- 🍽️ 餐食类型（日常、大餐）
- 🔄 一键恢复默认菜单
- 🔍 菜品搜索功能

### 3. 筛选设置
- 🕐 用餐时段筛选
- 💸 价格区间设置
- 🍴 餐食类型选择
- 👀 实时筛选结果预览

### 4. 数据统计
- 📊 吃饭次数统计
- 💵 消费分析
- 📈 菜品偏好排行
- 🕒 用餐时段分布
- 📝 历史记录查看
- 🗓️ 时间范围筛选

## 技术栈

- **框架**: UniApp (Vue 3)
- **样式**: SCSS
- **存储**: 本地缓存 (uni.storage)
- **图表**: 原生Canvas绘制
- **平台**: 微信小程序

## 数据结构

### 菜品数据结构
```javascript
{
  id: Number,           // 菜品ID
  name: String,         // 菜品名称
  tags: Array,          // 标签数组 ['早餐', '午餐', '晚餐', '夜宵', '零食']
  price: Number,        // 价格
  type: String          // 类型 ('日常' | '大餐')
}
```

### 历史记录结构
```javascript
{
  id: Number,           // 记录ID (时间戳)
  menuItem: Object,     // 菜品信息
  date: String,         // 日期 (YYYY-MM-DD)
  time: String,         // 时间 (HH:mm:ss)
  timestamp: Number     // 时间戳
}
```

### 筛选设置结构
```javascript
{
  tags: Array,          // 选中的标签
  priceRange: Array,    // 价格区间 [min, max]
  mealType: String      // 餐食类型 ('日常' | '大餐' | '')
}
```

## 使用说明

### 1. 抽签功能
1. 打开应用，进入主页
2. 可选择快速筛选标签（早餐、午餐等）
3. 点击"开始抽签"按钮
4. 等待抽签动画完成，查看结果
5. 可点击"再抽一次"重新抽取

### 2. 管理菜品
1. 切换到"粮库"标签页
2. 点击"添加菜品"新增菜品
3. 填写菜品名称、价格、类型和标签
4. 点击菜品可进行编辑
5. 可搜索和删除菜品
6. 点击"恢复默认"重置菜单

### 3. 设置筛选
1. 在主页点击右上角设置图标
2. 选择用餐时段标签
3. 设置价格区间
4. 选择餐食类型
5. 查看筛选结果预览
6. 保存设置

### 4. 查看统计
1. 切换到"统计"标签页
2. 查看总体统计概览
3. 选择时间范围（7天/30天/全部）
4. 查看菜品偏好排行
5. 分析用餐时段分布
6. 查看历史记录

## 默认菜单

项目内置了20个默认菜品，包含：
- 早餐类：小笼包、豆浆油条、煎饼果子
- 正餐类：蛋炒饭、麻婆豆腐、宫保鸡丁、红烧肉等
- 夜宵类：泡面、烧烤
- 零食类：薯片、奶茶、蛋糕
- 大餐类：火锅、海底捞、日式料理等

## 开发部署

### 环境要求
- Node.js >= 14
- HBuilderX (推荐)
- 微信开发者工具

### 运行步骤
1. 安装依赖：`npm install`
2. 使用HBuilderX打开项目
3. 运行到微信开发者工具
4. 在微信开发者工具中预览

### 发布步骤
1. 在HBuilderX中选择"发行 -> 小程序-微信"
2. 配置小程序相关信息
3. 生成发行包
4. 在微信开发者工具中上传代码
5. 提交审核
